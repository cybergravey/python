# Password Generator (Python)

## Overview
A simple password generator that creates a random password using letters, digits, and punctuation.

## What it demonstrates
- Functions with default arguments
- Random selection (`random.choice`)
- Building strings with `join`
- Using Python standard libraries (`random`, `string`)

## Requirements
- Python 3.x

## How it works
- Builds an “alphabet” of allowed characters
- Randomly selects characters for a requested length
- Prints the generated password

## Suggested Improvements (future)
- Allow custom length via user input or CLI arguments
- Toggle character sets (no symbols, digits only, etc.)
- Avoid ambiguous characters (O/0, l/1)
- Copy to clipboard (optional)
