# Keylogger (Lab / Educational)

⚠️ **Warning / Ethics**
This project is for **authorized, educational use only** (e.g., your own machine or a controlled lab).  
Do **not** use this on any device you do not own or have explicit permission to test.

## Overview
A minimal Python key capture demo using `pynput` to listen for keyboard events and write pressed keys to a local file.

## What it demonstrates
- Importing third-party modules
- Functions and event callbacks
- File writing (`open(..., "a")`)
- Basic exception handling
- Running a listener loop

## Requirements
- Python 3.x
- `pynput`

Install:
```bash
pip install pynput
